# 📊 Exploratory Data Analysis (EDA) on Retail Sales Data

## 📌 Project Overview

This project was completed as part of the **Oasis Infobyte Data Analytics Internship**.

The objective of this project is to perform **Exploratory Data Analysis (EDA)** on a retail sales dataset to identify customer purchasing behavior, sales trends, product performance, and generate actionable business insights using Python.

---

## 🎯 Objectives

- Perform initial data inspection
- Generate descriptive statistics
- Analyze monthly and quarterly sales trends
- Analyze customer demographics
- Analyze product category performance
- Visualize relationships between numerical variables
- Generate business insights and recommendations

---
