# 🧹 Task 3 - Data Cleaning

## 📌 Internship
Oasis Infobyte - Data Analytics Internship (OIBSIP)

## 📊 Project Overview
This project demonstrates the complete data cleaning process using the Titanic dataset. The dataset was cleaned and transformed into an analysis-ready format using Python and Pandas.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## 📂 Dataset
Titanic Dataset

## ✅ Data Cleaning Steps
- Loaded the dataset
- Generated a data quality report
- Identified and handled missing values
- Removed the Cabin column due to excessive missing values
- Filled missing values in Age using the median
- Filled missing values in Embarked using the mode
- Checked and removed duplicate records
- Standardized categorical values
- Verified and corrected data types
- Detected outliers using the IQR method
- Saved the cleaned dataset as a new CSV file

## 📁 Files Included
- `data cleaning.ipynb`
- `Titanic-Dataset (1).csv` (Original Dataset)
- `Titanic_Cleaned.csv` (Cleaned Dataset)
